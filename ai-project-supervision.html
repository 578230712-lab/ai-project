📋 第一步：配置腾讯文档

嵌入你的腾讯文档，团队成员可在网页中直接查看和编辑任务。

📌 没有文档？按模板创建：

  1. 打开 【腾讯文档】序号
  2. 点击「文件」→「生成副本」，创建你自己的表格
  3. 点击右上角「分享」→ 权限设为「获得链接的任何人可查看」
  4. 复制分享链接，粘贴到下方

💡 模板包含：任务名称、负责人、状态、优先级、开始日期、截止日期、完成进度、备注 共8列。

🔔 第二步：配置云之家机器人

配置 Webhook 后，逾期任务将自动推送到群聊并 @责任人。

📌 如何获取 Webhook 地址：

  1. 打开云之家 → 进入目标群聊
  2. 群设置 → 群机器人 → 添加机器人
  3. 复制 Webhook 地址

🚀 第三步：检查任务 & 推送通知

上传 Excel 文件（或从腾讯文档导出），系统自动检测逾期任务。

支持 .xlsx / .xls / .csv 格式。也可以从腾讯文档「文件 → 下载 → Excel」导出后上传。
将向云之家群聊发送消息，@所有逾期任务的责任人

🤖 进阶：下载自动化脚本

下载 Python 脚本，在你的电脑上设置定时任务，实现每日自动检查 + 自动推送。

脚本包含当前配置的云之家 Webhook，下载后按注释修改 Excel 路径即可使用。
